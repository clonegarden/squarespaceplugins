Anavo Tech · v1.0.0

Demo 1 — Vertical (default preset)

Company History

Vertical layout using the ?data= parameter. Scroll to watch the line draw and items fade in.

Install snippet:
timeline.min.js?layout=vertical&preset=default&data=[...]

Demo 2 — Alternate / Zigzag (elegant preset)

Our Story

Alternate layout using a hidden <ul> with data-year / data-title attributes. Items alternate left and right.

Install snippet:
timeline.min.js?layout=alternate&preset=elegant&divId=timeline-alt-list

Demo 3 — Horizontal (dark preset)

Product Roadmap

Horizontal layout. Scroll the timeline left-to-right on desktop; swipe on mobile. Line draws as you scroll.

Install snippet:
timeline.min.js?layout=horizontal&preset=dark&data=[...]
