💀 Logo Reaper

Logos enter from the right, get stamped at center, and pile up on the left.

🚀 Easy Install

One <script> tag in your Squarespace footer or code block.

🎨 Fully Configurable

Speed, size, words, colours – all set via URL query-string params.

♿ Accessible

Respects prefers-reduced-motion and disables heavy animations.

🔐 Licensed

Non-blocking license check – renders instantly, validates in background.

<!-- Squarespace Installation -->
<script src="https://cdn.jsdelivr.net/gh/clonegarden/squarespaceplugins@latest/
  logo-reaper/logo-reaper.min.js
  ?selector=%23logo-stage
  &height=220
  &logos=%5B%22https%3A%2F%2Fexample.com%2Flogo1.svg%22%5D
  &speed=90
  &bgColor=1a1a2e
  &stampColor=e94560
"></script>
