About This Demo

This demo shows the Tabbed Content plugin targeting a hidden <ul id="services-list"> element. Each <li> uses data-title, data-subtitle, and data-image attributes. The plugin extracts this data and renders a beautiful tabbed layout.

The script tag used below:

<script src="../tabbed-content.js?ulId=services-list&preset=elegant&debug=true"></script>

Try Other Presets

Replace preset=elegant with any of these in the script URL:

preset=default — Warm beige, serif headings, border separators preset=minimal — White background, no borders, clean sans-serif preset=elegant — Cormorant Garamond font, warm tones (shown above) preset=bold — Dark mode, white active color, bold typography
