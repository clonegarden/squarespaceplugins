About This Demo

Simulating a Squarespace 7.1 page with Image blocks

The Photo Grid plugin detects the section above by searching for a heading with the text "Photo Album". It collects every .sqs-block-image element, reads the native image dimensions from data-image-width / data-image-height, then removes the original blocks and replaces them with a fully responsive justified collage grid.
