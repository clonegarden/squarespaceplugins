Anavo Plugin Demo

Fixed Footer Reveal

The footer is hidden below the viewport and slides up when you scroll near the bottom of the page. Body padding is automatically added to accommodate it.

↓ Scroll to the bottom to see the footer reveal

01

Main Content Area

The plugin works with any footer element — native Squarespace footer, a custom footer section, or any element matching the selector. Keep scrolling.

02

No Layout Changes Required

The plugin dynamically fixes the footer and adds bottom padding to the body so nothing overlaps. When you scroll back up, the footer hides again.

03

Configurable Trigger

Adjust triggerOffset to control how many pixels from the bottom the reveal fires. Default is 200px. Works on all screen sizes.
