# Changelog

## [1.0.0] - 2026-02-19

### ✨ Initial Release

**Features:**
- Floating header animation (bottom → top)
- Universal Squarespace compatibility (7.0 & 7.1)
- Auto-detection of header and first section
- 15+ customization parameters
- Custom logo support
- Smooth CSS transitions
- Mobile responsive
- Commercial licensing integration

**Installation:**
- Zero-config installation
- CDN delivery via jsDelivr
- Auto-minification via GitHub Actions

**Parameters:**
- Styling: `fontSize`, `fontFamily`, `fontWeight`, `fontColor`, `bgColor`
- Logo: `customLogo`, `logoWidth`, `logoHeight`
- Behavior: `transitionSpeed`, `stickyTop`, `startAtBottom`, `zIndex`, `debug`
